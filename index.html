Features

Somes features

  • Get information about the bot
  • View and set the global translation language of the bot
  • Get information about items, tools, blocks, mobs, enchantment...
  • Translate an item
  • Display the list of mobs and bosses
  • Get the skin, name history, and UUID of a player
  • Get the list of Optifine versions
  • Send a random block

Commands

Information Commands

/bot

Allows you to have information about the bot.

/help

Allows you to have the list of orders available with the bot sorted into different categories.

/ping

Allows to ping the bot and see its latency ( â 100 ms).

Configuration commands

/lang

Allows you to see the languages available for the global translation of the bot and to define one for the server.

Minecraft commands

/recipe <item>

Allows you to display the recipe of an item.

Parameters

item - Name of the item requested in the languages included in the translate command.

/list <item>

Allows you to display the list of available recipes containing the requested item.

Parameters

item - Name of the item requested in the languages included in the languages included in the translate command.

/list <item id>

Allows you to search for the ID of an item, block or mob from the beginning of the item name.

Parameters

item_id - ID of the requested item.

/info <item>

Allows to have information on an item, a tool, a block...

Parameters

item - Name of the item requested in the languages included in the translate command.

/translate <item>

Allows to translate the requested item (does not take into account potions for the moment).

Parameters

item - Name of the item requested in the languages included in the translate command.

/mobs

Allows to display the list of passive, neutral, aggressive mobs and boss.

/mobinfo <entity>

Allows you to get information about the requested mob.

Parameters

entity - Name of the entity requested in the languages included in the translate command.

/skin <username>

Allows you to obtain the skin of the requested player.

Parameters

username - Requested player's nickname.

/names <username>

Allows you to see the history of a player's nicknames.

Parameters

username - Requested player's nickname.

/uuid <username>

Allows you to obtain the UUID of a player.

Parameters

username - Requested player's nickname.

/optifine

Allows you to have the list of Optifine versions.

/enchant <enchantment>

Allows you to have information about an enchantment.

Parameters

enchantment - Name of the enchantment requested in the languages included in the translate command.

/random

Allows to send a random block. For administrators it is possible to configure an automatic sending.
